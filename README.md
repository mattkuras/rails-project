# README

rails-project 

This a mini-blog web application that functions similarly to other platforms like Twitter and Reddit. Users sign up with identities. They are then put into communities where they can view posts specific to a chosen identity. Users can view all users' posts in their homepage, or they can make posts, comments, likes, among other features.


Prerequisites 

Github
Ruby 2.7.0
Rails 5.1.4



Install

Clone the repository
git clone git@github.com:mattkuras/rail-project.git
cd project
Install dependencies
Using Bundler 
bundle install



Usage

run rails s
navigate in browser to localhost:3000/
